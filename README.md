# 📊 Sales Performance Dashboard (2021–2024)

An interactive Microsoft Excel dashboard designed to analyze sales performance from **2021 to 2024**. Built using Pivot Tables, Pivot Charts, Slicers, and KPI visualizations, the dashboard helps users monitor sales trends, compare product performance, and gain actionable business insights.

---

## 📌 Project Overview

This dashboard transforms raw sales data into an interactive reporting solution, enabling users to explore sales performance across multiple dimensions such as year, month, product, company, payment method, customer age group, and city.

---

## 🎯 Business Objectives

- Monitor yearly and monthly sales performance
- Compare sales across products and companies
- Analyze customer demographics by age group
- Evaluate payment method preferences
- Identify top-performing cities
- Enable dynamic filtering for better decision-making

---

## 🚀 Dashboard Features

- 📅 Year-wise Sales Analysis
- 📈 Monthly Sales Trend
- 📦 Product-wise Sales
- 🏢 Company-wise Sales
- 💳 Payment Method Analysis
- 👥 Customer Age Group Analysis
- 🌍 Top 10 Cities by Sales
- 🎛 Interactive Slicers (Year, Quarter, Month, Brand & Age Group)
- 📊 KPI-driven Visualizations
